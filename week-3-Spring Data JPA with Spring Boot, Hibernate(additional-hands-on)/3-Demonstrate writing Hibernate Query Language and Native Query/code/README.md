# HibernateQueriesDemo

This project demonstrates:
- Hibernate Query Language (HQL)
- Native SQL queries

## Steps to Run
1. Import as Maven project into Eclipse.
2. Run `HQLAndNativeQueryDemo.java` as Java Application.
3. See console output.
